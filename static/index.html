Temporary hosting for static websites, usable programmatically.

With this service you can create a hosted website simply by uploading a zip or tar file containing your site (up to 50 MiB) via a HTTP POST. The below curl command will do just that:

curl --data-binary @website.zip https://tmpweb.net

The server will respond with the URL at which your website will be hosted for the next seven days.

/tmp/web.net is intended to aid in sharing generated HTML reports and design prototypes.

If you have any questions please contact admin@tmpweb.net.

The source code for this service can be found on GitHub.
